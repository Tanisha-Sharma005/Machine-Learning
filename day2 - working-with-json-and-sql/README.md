# 🧠 🔰 BASIC QUESTIONS

### 🔹 Q1. What is JSON?
**Answer:**  
JSON (JavaScript Object Notation) is a lightweight data format used for storing and exchanging data.

---

### 🔹 Q2. What is `read_json()`?
**Answer:**  
It reads a JSON file and converts it into a pandas DataFrame.

---

### 🔹 Q3. What is SQL?
**Answer:**  
SQL (Structured Query Language) is used to manage and query databases.

---

### 🔹 Q4. Why use SQL in ML?
**Answer:**  
Because most real-world data is stored in databases.

---

---

# ⚙️ 🟡 INTERMEDIATE QUESTIONS

### 🔹 Q5. What is nested JSON?
**Answer:**  
A JSON structure where values are themselves dictionaries or lists.

---

### 🔹 Q6. What is `json_normalize()`?
**Answer:**  
It flattens nested JSON into a tabular format.

---

### 🔹 Q7. What is `read_sql_query()`?
**Answer:**  
It executes SQL queries and returns data as a DataFrame.

---

### 🔹 Q8. Difference between SQL and pandas?
**Answer:**  
- SQL → used for querying databases  
- pandas → used for data manipulation  

---
